# Vector Database Automation Framework

## Overview

This module validates vector database operations used in Enterprise RAG systems.

## Test Coverage

- Connectivity
- Vector Insert
- Vector Search
- Similarity Search
- Metadata Filtering
- Top-K Retrieval
- Update
- Delete
- Index Validation
- Performance

## Supported Vector Databases

- PGVector
- Pinecone
- Milvus
- ChromaDB
- Weaviate
- Qdrant
- FAISS

## Execute

```bash
pytest vector_db/
```
