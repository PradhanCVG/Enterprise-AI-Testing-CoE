# API Automation Framework

## Overview

This module validates REST API endpoints used in Enterprise AI platforms.

## Test Coverage

- Health Checks
- Authentication
- Authorization
- Request Validation
- Response Validation
- HTTP Status Codes
- Error Handling
- Rate Limiting
- Performance
- End-to-End API Testing

## Supported Frameworks

- FastAPI
- Flask
- Spring Boot
- ASP.NET Core
- Express.js
- REST APIs
- GraphQL APIs

## Execute

```bash
pytest api/
```

Generate HTML Report

```bash
pytest api/ --html=reports/api_report.html
```
